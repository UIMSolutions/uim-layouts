# Tabler Badges

Badge components namespace for the Tabler layout package.

## Module

- `uim.styles.tabler.components.badges`

## Usage

```d
import uim.styles.tabler.components.badges;
```

## Status

This package currently provides the badges module namespace via `package.d`.
Concrete badge components can be added under this folder and re-exported from
`package.d` as they are implemented.
